Instructions

In this study, you will see picture and sentences pairs or pictures alone. You need to do two tasks:

  • Verifying whether the sentence correctly describes the content of the picture. If it does, press button Y (Yes); otherwise press button N (No). Please respond as quickly as possible.
  • Constructing a complete sentence to describe the picture with the given verb.

    • For picture verification task, you only need to verify whether the meaning of the sentence matches the picture. If a sentence is not grammatically correct but correctly describing the content of the picture, then you should press Y for yes.

    For picture description task, you will see a picture along with a verb under it. you need to ype in a full sentence using the given ver to describe the picture.
